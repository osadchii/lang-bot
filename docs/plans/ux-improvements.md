# UX-улучшения для изучения слов

## 1. Быстрое добавление слов

| Проблема | Решение |
|----------|---------|
| Много шагов для добавления карточки (выбор колоды -> метод -> ввод данных) | **Quick Add**: просто отправь слово в чат -> бот автоматически переводит и предлагает добавить в последнюю/избранную колоду |
| Нужно вручную указывать колоду каждый раз | **Колода по умолчанию**: настройка "моей колоды" для быстрого добавления |
| Нет возможности добавить несколько слов подряд | **Batch-режим**: после добавления карточки сразу предлагать "Добавить ещё?" вместо возврата в меню |
| AI-генерация показывает только результат | **Предпросмотр и редактирование**: показать AI-результат, дать возможность отредактировать перед сохранением |

## 2. Процесс обучения

| Проблема | Решение |
|----------|---------|
| Только 4 оценки (Снова/Трудно/Хорошо/Легко) | **Упростить до 3**: "Не помню" / "Вспомнил" / "Легко" — меньше когнитивной нагрузки |
| Нет подсказок при затруднении | **Кнопка "Подсказка"**: показать первую букву, часть слова или пример использования |
| Жёсткий лимит 20 карточек | **Гибкие сессии**: "Быстрая (5 мин)" / "Стандартная (15 мин)" / "Марафон" |
| Нет аудио произношения | **TTS интеграция**: кнопка прослушать произношение греческого слова |
| Показывается только греческое -> русское | **Обратный режим**: русское -> греческое для активного запоминания |
| Нет контекста запоминания | **Мнемоники**: AI-генерация ассоциаций для сложных слов |

## 3. Обратная связь и мотивация

| Проблема | Решение |
|----------|---------|
| Статистика только в отдельном разделе | **Мини-прогресс после каждой сессии**: "+15 слов выучено, streak 5 дней!" |
| Нет напоминаний | **Push-уведомления**: "У тебя 12 слов на повторение" в настраиваемое время |
| Нет геймификации | **Достижения**: "Первые 100 слов", "7 дней подряд", "Мастер артиклей" |
| Streak не визуализирован | **Календарь активности**: как в GitHub — зелёные квадратики по дням |
| Нет сравнения прогресса | **Недельный отчёт**: "На этой неделе +23% к прошлой" |

## 4. Организация слов

| Проблема | Решение |
|----------|---------|
| Только ручная организация в колоды | **Авто-теги**: AI автоматически присваивает теги (еда, глаголы, семья) |
| Нет поиска по словам | **Полнотекстовый поиск**: найти карточку по части слова |
| Нет "сложных слов" | **Leech-детекция**: автоматическая пометка слов, которые постоянно забываются |
| Нельзя отложить слово | **"Заморозить карточку"**: временно исключить из повторения |
| Дубликаты в разных колодах | **Умное предупреждение**: "Это слово уже есть в колоде X. Создать связь?" |

## 5. Контекстное обучение

| Проблема | Решение |
|----------|---------|
| Примеры не всегда есть | **Auto-примеры**: AI автоматически генерирует 2-3 примера для каждого слова |
| Нет связанных слов | **Семантические связи**: показывать синонимы, антонимы, однокоренные слова |
| Изучение изолированных слов | **Тематические мини-уроки**: "Заказ в ресторане" — 10 связанных слов + диалог |
| Нет грамматического контекста | **Грамматические карточки**: для глаголов — формы спряжения, для существительных — склонение |

## 6. Quick Wins (быстрые улучшения)

1. **Swipe-жесты** в веб-версии: свайп влево = "не помню", вправо = "помню"
2. **Keyboard shortcuts**: цифры 1-4 для оценок при обучении
3. **Undo**: отмена последней оценки ("Случайно нажал")
4. **Избранное**: пометить слово звёздочкой для приоритетного повторения
5. **Экспорт/Импорт**: CSV/Anki формат для миграции
6. **Тёмная тема**: для ночного обучения
7. **Офлайн-режим**: кэширование карточек для обучения без интернета

## Приоритеты внедрения

### Высокий приоритет (максимальный impact)

1. Quick Add — одно сообщение = одна карточка
2. Напоминания о повторении
3. Упрощённая оценка (3 вместо 4)
4. Кнопка "Подсказка"

### Средний приоритет

5. Обратный режим (русский -> греческий)
6. Leech-детекция проблемных слов
7. Достижения и streak-визуализация
8. Batch-добавление

### Низкий приоритет (приятные дополнения)

9. TTS произношение
10. Тематические мини-уроки
11. Офлайн-режим
